# Nintendo Switch Joy-Con as a mouse for macOS

You can move a mouse pointer with Nintendo Switch Joy-Con and this program.

### Prerequisites

* macOS Mojave 10.14.6
* Xcode 11.3.1
* Nintendo Switch Joy-Con
